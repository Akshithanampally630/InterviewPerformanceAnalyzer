🎤 AI Interview Answer Analyzer
An intelligent web app that analyzes interview answers and provides feedback using AI-powered techniques like semantic analysis, sentiment detection, filler word detection, and structured response evaluation.

🚀 Features
✅ Semantic Matching – Checks how well your answer matches the question
😊 Sentiment Analysis – Evaluates tone and confidence
🗣 Filler Word Detection – Identifies unnecessary words (um, uh, etc.)
⭐ STAR Method Scoring – Evaluates structured answers (Situation, Task, Action, Result)
📊 Confidence Score – Overall performance rating out of 100
✨ AI Improved Answer – Enhances your response professionally
🧠 AI Summary – Summarizes your answer in 2–3 lines
🎤 Speech-to-Text – Convert spoken answers into text
🛠 Tech Stack
Frontend: Streamlit

Backend: Python

AI Model: Gemini API

Libraries:

transformers
nltk
speech recognition (if used)
